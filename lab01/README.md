Програма аналізує безпеку пароля з урахуванням персональних даних користувача (ім’я, дата народження). Програма також надає рекомендації для покращення пароля.

----------------------------------------------------Використовується система оцінювання за вагою категорій (0–10 балів):----------------------------------------------------

Структура (0–4 бали)
1.Великий та малий регістр
2. Цифри
3. Спецсимволи
4. Використано >=3 типи символів

Безпечність (0–3 бали)
1. Відсутність імені
2. Відсутність дати народження (день, місяць, рік)
3. Відсутність простих шаблонів (1234, qwerty, password, abcd)

Довжина (0–3 бали)
1. <8 символів — 0
2. 8–11 символів — 1
3. 12–15 символів — 2
4. 15 символів — 3

Рівень безпеки:

0–3 — Слабкий
4–6 — Середній
7–10 — Сильний

----------------------------------------------------ВСТАНОВЛЕННЯ ТА ЗАПУСК----------------------------------------------------

1. Скачайте файл main.py.
2. Встановіть Python (версія 3.7+).
3. Відкрийте термінал/командний рядок.
4. Перейдіть у папку з файлом:
   cd path/to/folder
6. Запустіть програму:
   python main.py (АБО через RUN)
